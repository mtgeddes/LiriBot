# LiriBot
LiriBot is an app that shows the use of APIs within a CLI.

After running `npm i`, begin app by running "node liri" from the command line. It will list the available options.

* [Spotify](#spotify)
* [Twitter](#twitter)
* [OMDB](#omdb)
* [Random](#random)
